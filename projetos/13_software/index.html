O que é um programa/software

É um conjunto de declaração (Statements)

Statements são conjuntos de expressões

Software tem um 'Objetivo'

Statements 'guiam' o sofware para seu objetivo

Os statements e expressios de fomas simples

Statements:console.log(1>2)

Expression:console.log('Mitologia Grega')

E é essa a junção dos conceitoa!
